# Interest form

> itDAGENE interest form

### Development
```zsh
# Install dependencies
$ yarn

# Concurrently run the FORM and the API
$ yarn start
```

### Build and run image
```zsh
$ docker build -t <TAG> .
$ docker run -p 3000:3000 -p 8000:8000 -it <TAG>
```

### Using docker-compose
```zsh
$ docker-compose up -d
```
